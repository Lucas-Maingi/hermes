# Hermes — WhatsApp AI Commerce & Support Agent (Live Demo)

This Hugging Face Space runs the Hermes chat simulator + owner dashboard,
seeded with demo data.

- Chat with the agent (English or Swahili/Sheng — try `nataka 2 unga na 1 cooking oil`).
- Press the **enter M-Pesa PIN** button to run a payment end-to-end.
- Watch the **owner dashboard** update (deflection rate, orders, revenue).

The demo runs on the built-in simulators. The same code runs on the real
Meta WhatsApp Cloud API and M-Pesa Daraja when credentials are set.

- **Source code & full documentation:** https://github.com/Lucas-Maingi/hermes
